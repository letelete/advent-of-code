## How to run?

You can execute each solution and its tests on your own; feel free to do so :nerd_face:

\*`{NUMBER}` - A number of the day to execute

### Running solutions

`npm start --day={NUMBER}`

example: `npm start --day=8`

### Running tests

`npm test --day={NUMBER}`

example: `npm test --day=10`
