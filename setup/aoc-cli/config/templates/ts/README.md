# TypeScript AoC Template

A simple template for solving Advent of Code (AoC) challenges using TypeScript.

## Usage

Run the following command to execute the solver. The output will display the answers to both parts of the challenge, each on a new line.

```bash
$ pnpm start
```

#### Tests

Run the following command to execute the test runner. This will run all samples from the [in.samples.yml](./in.samples.yml) file.

```bash
$ pnpm test
```

### Example

```bash
$ pnpm start

1320851
26859182
```

### Options

```bash
--verbose (-v)
```

Use the `--verbose` flag to enable detailed output, including additional information during execution.

```bash
$ pnpm start --verbose
```
